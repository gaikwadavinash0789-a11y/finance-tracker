# finance-tracker
A modern and responsive Finance Tracker

Finance Tracker is a responsive web application designed to help users manage their personal finances efficiently. It allows users to add and track income and expenses, view transaction history, and monitor their net balance in Indian Rupees (₹). The application provides a clean and user-friendly interface for better financial management.

Skills / Technologies Used
HTML5 – For structuring the web pages.
CSS3 – For styling and creating a responsive user interface.
JavaScript (ES6) – For implementing application logic, handling transactions, and updating financial data dynamically.
Project Features
✅ Add and manage income and expenses.
✅ View transaction history.
✅ Automatic net balance calculation.
✅ Responsive and user-friendly design.
✅ Real-time updates of financial data.
